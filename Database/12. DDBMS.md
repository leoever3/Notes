### DDBMS

Distributed database

logically related data in two or more physically independent sites

connected via computer network



Distributed processing

division of logical database processing among network nodes



Distributed databases require distributed processing



Main components of DDBMS

transaction processor  data processer



SPSD. MPSD. MPMD



Homogeneous distributed database system

Heterogeneous distributed database system



DDBMS characteristics are a set of transparencies



Transaction is formed by one or more data requests



Distributed concurrency control is required in network of distributed database



DDBMS evaluates every data request



Design of DDBMS must consider fragmentation and replication of data



Database can be replicated over several different sites on computer network
